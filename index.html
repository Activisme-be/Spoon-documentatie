      # Spoon documentatie

      Alle nodige documentatie om met de Laravel gebaseerde template genaamd Spoon aan de slag te gaan.

      # Contributie richtlijnen

      Als u documentatie indient voor de huidige stabiele release, dient u deze in bij de overeenkomstige branch toe te voegen. Documentatie voor Spoon 1.0 zou dus bijvoorbeeld moeten worden toegevoegd in de 1.0 branch. Documentatie bedoeld voor de volgende release van Spoon moet worden ingediend bij de master branch.
